Git Tools Documentation
=======================

**Git Tools** is a comprehensive suite of Bash/Shell utilities designed to streamline and enhance your Git workflow. This collection provides both simple wrappers for common Git commands and complex composers for advanced Git operations.

Features
--------

* **Simple Wrappers**: Quick aliases for frequently used Git commands
* **Advanced Composers**: Complex workflows for branch management, repository creation, and more
* **Multi-Repository Support**: Integration with myrepos for managing multiple repositories
* **Comprehensive Testing**: BATS-based testing framework with CI/CD integration
* **VCSH Integration**: Version control for your home directory dotfiles
* **API Integration**: Create repositories via GitHub and GitLab APIs
