# Git tools repository

## Objective
bash **configs** and **programs** to seamlessly interface with my git repositories

## Configurations
### git
`.gitconfig`
`.gitignore`
`.gitmodules`

### mr
`.mrconfig`
`.config/mr/available.d`
`.config/mr/config.d`

## Programs
`bin/*.sh`

### Classes  
* wrappers for preferred **singleton** git command implementations
	* e.g, `gl.sh`, `glvv.sh`
* routines for custom task that involves **multiple** git commands
	* e.g., `git_rebase_local_remote.sh`
